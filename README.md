This is a simple Express-based API for managing a movie collection. It allows you to perform CRUD (Create, Read, Update, Delete) operations on movie data stored in a MongoDB database.

Features :
1) Create a new movie
2) Retrieve all movies or a specific movie by ID
3) Update movie details
4) Delete a movie
5) Display movie data in an HTML list
Tech Stack
Backend:
Node.js,
Express.js
Database:
MongoDB
ODM: Mongoose
